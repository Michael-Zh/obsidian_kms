## Builds Completed This Session

### 1. AutoSleep → sleep_log.md (iOS Shortcut automation)
- Shortcut runs daily at 12:00, appends AutoSleep data to `04_project/Training_program/sleep_log.md`
- Format: block per day with Time Asleep dictionary + Readiness dictionary + blank Note field
- Uses Append to File (not prepend — iOS limitation)
- Archive plan: sleep_log.md archived at each monthly review session alongside Fitbod CSV processing

### 2. Morning Priming Skill (`/priming`)
- Skill file created at `~/.claude/skills/priming/SKILL.md`
- Input: Typeless voice transcript via WeChat → Claude Code
- Context loaded: POS + priorities + sleep_log + Google Calendar (both training + main) + most recent coaching_session + coaching_discussion
- Output: (A) daily note block with main goal / top 3 tasks / schedule / energy / POS nudge + HTML link; (B) self-contained HTML card file in `03_output/priming/priming_YYYYMMDD.html`
- Tone: warm, energising, friend-who-knows-you — not productivity robot
- Working hours rule: Mon–Fri 09:00–17:30; work tasks stay in window
- Music layer deferred to V2

### 3. Google Calendar `--today` flag
- Added to `calendar_scheduler_v3.py` — fetches both training calendar + main calendar events for today
- Returns clean JSON: `{date, training_events: [...], main_events: [...]}`
- Tested and confirmed working

### 4. Laptop wake automation
- `sudo pmset repeat wakeorpoweron MTWRF 07:00:00` — Mac wakes at 07:00 weekdays only (user ran manually)
- launchd agent created: `~/Library/LaunchAgents/com.michael.caffeinate.plist` — runs `caffeinate -i` on login, keeps Mac awake until manually cancelled
- Agent loaded and running (PID confirmed)
- To stop: `launchctl unload ~/Library/LaunchAgents/com.michael.caffeinate.plist`
- To restart: `launchctl load ~/Library/LaunchAgents/com.michael.caffeinate.plist`
