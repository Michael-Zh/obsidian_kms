## 背景与初衷

Obsidian 是一款非常优秀的笔记软件，但由于其基于 Markdown 语法的特性，在不装插件的情况下，图片仅能进行缩放，无法像图文字那样快速调整对齐方式，拖拽体验也存在问题。

作者期望的排版功能更接近飞书云文档：支持多媒体分栏放置、拖拽改变位置，以便在做可视化笔记时更加方便。因此开发了一款 Obsidian 插件，让媒体排版体验接近飞书。

插件由作者借助 AI 完成开发（无编程基础）。使用前**强烈建议先备份 Obsidian 仓库**，确保文件安全。

## 核心功能

-   从本地拖拽图片或视频进入笔记时，媒体会自动转换成一个可进行排版操作的**布局块**。
-   如果笔记原本就存在图片/视频，加载插件后没有被自动转换，可以单击或拖拽一下该媒体，让插件检测到后即可转换。
-   布局块内的单个媒体支持拖拽右下角调整大小，并可拖拽改变排列顺序或拆分分栏。
-   整行显示的媒体支持右键选择**居左 / 居中 / 居右**三种对齐方式。
-   当媒体下方没有文字时，拖入新的媒体会自动与同行媒体分栏显示。
-   每行最多支持 **4 栏**显示，第 5 个媒体文件会自动放在下一行。
-   同行媒体块高度自动保持一致。
-   将鼠标放在分栏图片上时，其右侧会显示一条线条，按住可改变该图片的行宽。
-   不同章节（文字段落分隔的区域）之间的媒体可以相互拖拽；若目标区域为空，则需先插入一张媒体让其转换为布局块，或使用插件命令新建空布局块后再拖拽。
-   右键媒体可查找该文件在目录中的位置。
-   右键媒体可为图片**添加文字注释**，注释同样支持居左 / 居中对齐。
-   **视频文件同样适用**以上分栏、拖拽、文字注释、对齐等功能。

## 下载与安装

### 下载地址

-   主下载地址：**GitHub**。
-   若 GitHub 登录不便，作者也提供了一个备用下载网站。

### 安装步骤

1.  下载插件后，将 **三个文件**放到仓库下的指定目录（一般为 `.obsidian/plugins` 下对应的插件文件夹）。
2.  打开 Obsidian 设置 → **第三方插件 → 关闭安全模式**。
3.  刷新插件列表，找到该插件并**启用**。
4.  若笔记里已有的图片未自动转换为布局块，**手动拖拽**一下原图，插件即可识别并转换。

## 重要命令

插件提供两条关键命令，使用 Command + P 调出命令面板执行：

-   **Visual Media Layout: Restore all vault layouts to normal media**  
    将整个仓库内的所有媒体恢复为 Obsidian 原版显示方式。_关闭插件前请务必先执行这条命令_，否则用于排版的布局块会变成一串代码，图片或视频将无法正常显示。
-   **Visual Media Layout: Enable automatic media layout conversion**  
    重新开启布局块排版模式。在执行过上一条"还原"命令后，如想再次使用插件排版功能，需执行该命令。

## 扩展与后续

当前插件功能刻意保持精简，目的是保留 Obsidian 以文本为主的编辑模式。若希望在此基础上扩展更多功能，可在 GitHub 下载资源包后，借助 AI 自行丰富。

## 注意事项

-   插件由 AI 协助开发，可能存在因技术知识不足引发的潜在问题，安装前请**务必备份 Obsidian 仓库**。
-   关闭插件前一定要先执行"还原所有布局"命令，避免媒体被转换为代码导致无法查看。
-   插件同时支持本地图片与视频文件。
