# 这是一篇写给"也想搬但不知道怎么搬"的你的文章。

也是我给自己留的一个记录。

---

## 一、我和印象笔记的14年

2012年，我第一次注册Evernote国际版。

那时候正在读博，刚开始认真对待"记录"这件事。从第一条笔记到现在，差不多14年过去了。这些年，我在印象笔记里积累了6000多条笔记：

-   将近500条**深度察觉日志**——每天记录内心状态、属灵操练、情绪盘点的地方
    
-   2000多条**微信文章收藏**——十年来看过的、觉得重要的文章，全存在这里
    
-   1000多条**科研教学资产**——研究笔记、文献综述、论文草稿、学生档案、课程设计
    
-   当妈以后，还有上千条的**个人和家庭成长记录**——育儿课程学习笔记、个人成长读书笔记……
    

毕业回国之后，切换到印象笔记国内版。参考图书馆分类系统，我搭了一套自己的笔记本结构，付费订阅付到了2030年。

这不是一个普通工具。它是我14年思考和生活的容器。

---

## 二、为什么要离开

前两年，我开始觉得印象笔记越来越卡。

不是一夜之间的决定。是慢慢地，越来越有一种无力感。

最核心的两个问题。

**第一，笔记不能批量导出。**

我的笔记，我想带走，印象笔记说：不行。或者说不是完全不行，是非常麻烦，要一个笔记本一个笔记本地手动导。即使是打印，几篇文章也只能一篇一篇地打，不能合并成同一篇文件，一键打印。一个想拿回自己数据的人，会不断撞上一堵看不见的墙。

**第二，无法被AI调用。**

今年四月，我开始搭建自己的LLM WIKI，真正开始把AI融入学习和创作的流程。我意识到，笔记的价值不只是"存"，更是能被调用，能和AI对话，能成为你知识系统真正活着的一部分。

印象笔记做不到。它是一个封闭的孤岛。

两件事加在一起，我做了决定：把数据搬走。

---

## 三、为什么是Obsidian

今年四月，我开始试用Obsidian。

开始只是试试。但在搭建LLM WIKI的过程中，我越来越确定一件事：

**Obsidian是AI时代最丝滑、最安全的笔记软件。**

原因很简单：

-   文件是Markdown，存在本地，是你的
    
-   可以被AI直接读取和调用
    
-   开放生态，不被平台绑架
    
-   没有围墙
    

用了几个月之后，我发现自己已经不想回去了。

但那6000多条笔记，还锁在印象笔记里。好像被龙看守的金矿，深埋地下。

---

## 四、24小时的故事

想把笔记搬出来，这个念头在我脑子里转了很久。

问题是：印象笔记没有一键批量导出的方法。

直到我找到一个工具：**evernote-backup**。

这是一个开源的命令行工具，专门把印象笔记备份为ENEX格式。我用Claude Code帮我跑了整个流程。

我是文科生，命令行对我来说是陌生的东西。但Claude Code让这件事变得可以完成——我只需要告诉它我想做什么，它帮我写命令，帮我处理报错，帮我盯着进度。

过程比预想的复杂一点。

印象笔记的API有频率限制：每隔一段时间只能下载一批笔记，超过就会被暂停，要等几分钟到几十分钟不等。6000多条笔记，分成十几批，断断续续跑了将近24小时。

中间还遇到进程卡死——是因为电脑休眠把进程挂起了。发现之后手动唤醒，继续跑。

但最后，全部下完了。

打开Obsidian，看到14年的笔记按原来的笔记本结构整整齐齐排在那里，我有一种很踏实的感觉。

说不清楚为什么。大概是因为，这些东西，现在真的是我的了。

## 五、完整操作步骤

如果你也想做这件事，下面是完整方法。

**你需要准备的工具**

-   evernote-backup（Python命令行工具，免费开源）
    
-   印象笔记开发者Token（免费申请）
    
-   Obsidian + Importer插件（官方插件，免费）
    
-   一台不会频繁休眠的电脑，以及耐心
    

---

**第一步：安装 evernote-backup**

打开终端，输入：

```
pip install evernote-backup
```

---

**第二步：获取开发者Token**

浏览器登录 app.yinxiang.com，然后访问这个地址：

```
https://app.yinxiang.com/api/DeveloperToken.action
```

点击生成Token，复制备用。

> 注意：很多人的账号密码登录是被禁用的（尤其是用微信或手机号注册的），必须用Token方式登录。

---

**第三步：初始化**

```
evernote-backup init-db --backend china --force --token 你的Token
```

看到`Successfully initialized database`就成功了。

---

**第四步：同步笔记**

这一步最花时间。Claude Code 写了一个自动重试的脚本，遇到限流自动等待，不用一直盯着：

```
cat << 'SCRIPT' > /tmp/evernote_sync_loop.sh
#!/bin/zsh
whiletrue; do
echo"=== 开始同步 $(date) ==="
  output=$(evernote-backup sync 2>&1)
echo"$output"
ifecho"$output" | grep -q "Synchronization completed"; then
    echo"=== 全部完成！==="
    break
fi
  wait_time=$(echo"$output" | grep "Restart program in" | grep -oE '[0-9]+:[0-9]+' | tail -1)
if [ -z "$wait_time" ]; then
    echo"=== 60秒后重试... ==="
    sleep 60
else
    minutes=$(echo"$wait_time" | cut -d: -f1)
    seconds=$(echo"$wait_time" | cut -d: -f2)
    total_seconds=$(( minutes * 60 + seconds + 30 ))
    echo"=== 限流，等待 ${total_seconds}s ==="
    sleep $total_seconds
fi
done
SCRIPT
chmod +x /tmp/evernote_sync_loop.sh
/tmp/evernote_sync_loop.sh
```

**一定不要让电脑休眠**——这是血泪教训。开着屏幕，或者设置电源不休眠。

笔记越多，等待时间越长。6000条大概需要8-24小时。

---

**第五步：导出为ENEX文件**

```
mkdir -p ～/evernote_export
evernote-backup export ～/evernote_export/
```

几分钟后，你会在`～/evernote_export/`看到按笔记本结构整理好的`.enex`文件。

---

**第六步：导入Obsidian**

1.  在Obsidian安装**Importer**插件（Settings → Community plugins → 搜索Importer）
    
2.  `Cmd+P`→ 搜索**Import data**
    
3.  选择**Evernote (.enex)**
    
4.  选择`～/evernote_export/`文件夹
    
5.  点Import，等它跑完
    

完成之后，笔记就在Obsidian里了。原来的笔记本结构还在，图片和附件也都在。

## 六、我真正想说的

这24小时，表面上是技术迁移。

但对我来说，它有一个更重要的意义：**把自己14年的积累，真正拿回自己手里。**

在AI快速发展的今天，我越来越在意一件事——我的数据，我做得了主吗？

笔记不只是存信息的地方。它是你思考的轨迹，是你生命的一部分。如果这些东西只能锁在一个封闭的平台里，无法被你自己的AI系统调用，无法被你自己掌控……那它的价值是打折的。

Obsidian的底层逻辑不一样。

你的笔记是你的文件，存在你的电脑上，可以随时搬走，可以接入任何工具，可以被AI直接读取。没有平台绑架，没有围墙。

这件事叫做**笔记主权**。

---

还有一件事，关于人的所是。

当AI可以低成本替代越来越多的工作，当你在某个分工环节上创造价值的机会被慢慢拿走，很多人会陷入一种困惑：

我是谁？我要干什么？我在乎什么？什么对我来说真正重要？当AI可以代劳工作，我还能创造什么独特的价值？

这些问题的答案，我越来越相信，有很大一部分，就藏在我们一路走来的日记、笔记、思考和复盘里（年计划社群的小伙伴们，如果你们跟着JP姐姐复盘了三年四年，你现在手里的关于自己的信息资产是不可估量的）。

AI时代是马太效应时代。你有的，还要加给你。你没有的，连你有的那些也要夺去。

我那些关于自己的记录——将近500条深度察觉日志（还不算几十本手写日志），十四年的思考痕迹，那些关于我的高山低谷，我的来路归途，那些数算恩典的文字，在暗夜中的仰望和等候，心如刀割或是喜不自禁，都刻画了我的起初所是，我会被用于何事，我终将归属谁氏——这些重要又宝贵的信息，在印象笔记里静静等待了十几年。一直在累积，从未被真正调用。

现在，正是借助AI让它们发挥价值的时候。

这些笔记里，藏着最真实最原始的JP。AI的出现，让这些文字有了全新的可能性——它可以借给我一双眼睛和超级智能，帮助我识别行为模式、追踪成长轨迹，追溯封存的记忆。AI+记录，可以深度和广度上协助解锁我自己。

但第一步，是把这些宝贝从层层上锁的府库里搬出来。

AI时代，信息越来越便宜。但你真正拥有的东西——你的思考、你的察觉、你的记录——它们不只值得被好好保管，更值得被真正用起来。
