5月内测，6月8日全量上线。RED Skill 是小红书今年最值得关注的更新——创作者把经验封装成 SKILL.md 挂在笔记下面，用户一键复制到 Claude Code 就能跑。以前在小红书种草内容，现在种草工具。  
这次整理了 8 个目前生态里最值得关注的 Skill：  
🛠️ guizang-social-card-skill  
归藏出品，28种版式 + 10套主题，AI 直接生成小红书图文卡和公众号封面。3400+ 人已经在用，搜"小红书 AI 作图教程"底下大概率挂着它。  
⭐ baoyu-skills  
16K GitHub Star，自媒体人专属视觉工具箱。图文卡片、信息图、封面、配图，一套全搞定。  
✍️ xhs-writer-skill  
爆款笔记生成器，基于 gpt-image-2。图生图卡片 + 5种标题公式（利诱型、反常识型、情感共鸣型），支持自然语言修改。  
📊 xiaohongshu-ops-skill  
从内容生产升级到账号运营。推荐流分析、自动发布、爆款复刻、评论回复。有用户搭了"一人三号"方案，14天跑通。  
📝 ai-content-skills  
核心差异化：内置反AI味词库。解决"读起来像AI写的"这个最让人头疼的问题。  
🚀 chenjin-cmd  
微信+小红书+抖音+Twitter 四平台起号方案，一个 Skill 覆盖全栈分发。  
🎬 oil-cover  
AI工具实操视频封面，垂直定位，干净精致。  
⚙️ skill-workflow  
五步流水线：爬取→清洗→分析→创作→格式化。基于 Claude Agent SDK，零经验半小时搭一套内容生产系统。  
一个观点：AI Skill 的竞争拼的不是技术，是分发效率。谁能让不会写代码的人最快用上 AI，谁就赢。小红书用"种草"逻辑做 AI 工具分发，触达效率比 GitHub 高一个量级。
