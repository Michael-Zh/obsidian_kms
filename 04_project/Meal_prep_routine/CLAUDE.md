# Project Context: Meal_prep_routine
_Last updated: 2026-07-21_

## What This Project Is

可持续饮食系统的设计——回答三个核心问题：**什么时候吃、吃什么、吃多少**。这是 body recomposition 的基础层。当前为独立项目（P2），先讨论哲学 + routine 设计，之后与 Training Program 结合并集成到 Training Coach App。

**Primary Pillar:** PhysicalHealth  
**Status:** active — Phase 1: philosophy + routine 设计  
**Current Focus:** 讨论 eating philosophy、fasting window redesign、可重复的 meal prep 流程  
**Priority:** P2  
**Target Completion:** 2026-09-30

---

## Objectives

1. **设计可持续的饮食 routine** — 回答什么时候吃、吃什么、吃多少
2. **解决 16:8 执行困难** — 重新设计 fasting 窗口或调整为更可行的方案
3. **建立可重复的 meal prep 流程** — IKEA container 采购、batch cook 流程
4. **最终集成到 Training Coach App** — 成为 LMS 执行层的一部分
