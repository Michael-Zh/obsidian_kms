# Scheduling Rules — Danseur Noble Training Program
_Last updated: 2026-06-02_

## How to Use This File

These rules are used to:
1. Build the default weekly schedule
2. Reschedule when life intervenes (dinners, shows, travel, low energy)
3. Flag conflicts before confirming a week
4. Determine when a deload week is due

Reference the Class Pool (`Training_program_Class_Pool.md`) for available options and intensity tags.

---

## Rule 1 — Weekly Volume Target

**Normal week:**
- Gym: target 2x (Mon back/legs/arms + Tue chest/shoulder), **minimum 1x**. Rescheduling fallback: Mon+Thu or Tue+Thu if one primary slot is blocked.
- 2x Ballet: Sun 16:00 ADC (primary) + Thu 17:30 ADC or HJS (backup/flexible)
- 1x Jazz (weeks 2 + 4 of choreo cycle; see Rule 9)
- 1x Iyengar Yoga — default Mon evening after gym (see R2a)
- 1x Tue evening: ADC Improv Contemporary 18:45 *(temporary arrangement — regular Tue dance classes on hold)*
- 1 full rest day minimum (Saturday protected)
- **At least 1 intentional double session per week** (stamina target)

**Total active days: ~5–6. Total High-intensity sessions: max 4.**

> ⚠️ With the new structure (Sun ballet → Mon gym+iyengar → Tue gym+optional light → Wed jazz → Thu ballet), high days cluster on Sun+Mon+Tue+Thu. If Friday Mysore is scheduled, consider swapping Thu ballet for Iyengar to stay within the 4-high cap.

### R2a — Monday Sequencing Rule
Monday is a structured recovery day after Sunday ballet: afternoon gym (back/legs/arms) followed by evening Iyengar for stretch and alignment.

**Sequence:** Office → Gym ~15:00 (~16:30) → Iyengar 20:15 (~21:15) → Sauna/Sleep prep

This pairs one High (gym) with one Light (iyengar), not a double-High. Compensating rules:
- Sunday ballet is the only load before Monday — no prior double to recover from
- Tuesday gym (chest/shoulder) uses different muscle groups from Monday — spacing is safe
- If 3+ consecutive low-energy mornings → drop Iyengar on Mon; just do gym or skip both

## Rule 3 — Mysore Placement

**Default slot: Friday 07:00**
- Reasoning: cleanest day for it — no office, Thu evening is already light, no consecutive doubles conflict
- If Friday Mysore → Friday is Mysore only; no evening class that day

**Backup slot: Thursday 07:00** (last resort only)
- Only use if Friday is unavailable (travel, early commitment)
- If Thu Mysore: evening must be Light only (Iyengar or Yin); no exceptions
- Thu Mysore means 3 consecutive High days (Tue → Wed → Thu) — watch energy carefully

**Avoid: Tuesday and Wednesday Mysore**
- Tuesday: same day as Tue gym (chest/shoulder); chaturanga repeats shoulder/chest muscles within hours — unsafe overlap
- Wednesday: office day + possible jazz+hiphop evening; adding 07:00 Mysore creates excessive load

## Rule 4 — Muscle Group Spacing

| Session | Primary muscles | Min gap to next similar |
|---------|----------------|------------------------|
| Sun Ballet | Legs, core, posture chain | Low overlap with Mon gym (different focus) -> OK |
| Mon Gym (back/legs/arms) | Back, glutes, hamstrings, biceps | Tue gym uses different groups -> OK |
| Tue Gym (chest/shoulder) | Chest, shoulders, triceps | 72h before Mysore (chaturanga) -> Fri OK |
| Mysore / Ashtanga | Chest, shoulders (chaturanga), full core | 48h after Tue gym -> no Wed Mysore; Fri is safe |
| Ballet | Legs, core, posture chain | Can follow any gym day fine |
| Jazz / Hiphop | Cardio, legs | Low overlap with gym; fine anytime |

**Key constraint:** Tue gym (chest/shoulder) -> Mysore must be Fri or later, not Wed or Thu. Fri Mysore at 07:00 provides a ~65h gap -- safe.

## Rule 5 — Rest Day

- **Saturday = always rest.** No classes, no gym. Nervous system reset before Sunday ballet anchor.
- Exception only if Sunday ballet is moved to another day that week (rare).
- Light walks, stretching at home, sauna alone are fine — no structured classes.

## Rule 6 — Deloading

### Cadence
- **Default: every 5 weeks**
- Can shorten to week 4 if: energy score <3/5 for 3+ consecutive days, or heavy social/show week ahead
- Can extend to week 6 if: energy score consistently 4+/5 for the past 2 weeks
- **Hard limits: never sooner than 3 weeks, never later than 6 weeks**

### Trigger Process
I will flag the upcoming deload 1 week in advance and propose the schedule. You confirm or push by 1 week if energy is genuinely high, choosing Variant A or B based on the choreo cycle.

### Deload Week Template

Two variants determined by choreo cycle:

**Variant A — Wed Iyengar (non-choreo weeks: Jazz weeks 1+3)**

| Day | Activity | Notes |
|-----|----------|-------|
| **Sun** | ADC Ballet | Keep 1x ballet — single class, no stack |
| **Mon** | Full body gym — half weight, half exercises | Reduce volume; keep the movement pattern |
| **Tue** | Reformer Pilates | Core + control without impact |
| **Wed** | Iyengar 18:30 | Replace Jazz with alignment work |
| **Thu** | Yin Yoga | Deep passive holds |
| **Fri** | Rest (flexible: Reformer if good teacher) | Default rest; teacher override allowed |
| **Sat** | Rest | Protected |

**Variant B — Wed Jazz (choreo weeks 2+4)**

| Day | Activity | Notes |
|-----|----------|-------|
| **Sun** | ADC Ballet | Keep 1x ballet |
| **Mon** | Iyengar Yoga | Replace gym with alignment/stretch |
| **Tue** | Full body gym — half weight, half exercises | Reduce volume |
| **Wed** | ADC Jazz 19:00 | Keep key choreo week attendance |

**Deload week total:** 1x ballet + 1x full body gym (half) + 1x iyengar + 1x reformer + 1x yin
No Mysore during deload week. No HJS. No double sessions. Reformer slots interchangeable by teacher availability.

## Rule 7 — Rescheduling When Life Intervenes

### Dinner / Social Plans on a Class Night
1. Check if a morning option is available that day (HJS, Mysore) — if yes, swap
2. If no morning option → treat as rest day
3. **Never cancel Sunday ballet for a social event.** Reschedule the social, or attend an earlier ballet class that day if available.
4. Ballet takes priority over Contemporary/Hiphop when choosing what to keep

### Show / Performance Night
- Evening class = rest; don't try to squeeze gym after a show
- If show is Thu → Yin/Iyengar moves to Fri; Mysore slot shifts to Fri 07:00
- 2+ consecutive show nights = mini deload: keep only gym and one Light session those days

### Good Weather / Sunny Days
- Sunny weather is a valid reason to replace indoor sessions with outdoor time — no guilt, no makeup required
- Priority: outdoor movement (walk, cycle, skate, sun) > any indoor class that day
- If the whole weekend is sunny → protect Saturday rest but swap Sunday ballet to an outdoor activity; resume normal week on Monday
- Multi-day sunny stretch (2+ days): treat like a mini travel week — keep at most 1 structured session per day, rest is outdoor recovery

### Public Holidays
- Treat like a rest day or outdoor day depending on weather
- Do not reschedule displaced sessions to compensate — just absorb the gap and resume
- Gym anchor only if gym access confirmed at destination; otherwise replace with Mysore or Yin
- No doubles on travel days (one session max)
- Return day = rest or Light only

### Low Energy Signal (3+ days of morning score <3/5)
- Drop the next High session to its Moderate/Light alternative
- Flag whether deload week should be triggered (see Rule 6)
- Do NOT push through — the energy crash pattern is documented in [[Fitness-Routine]]

## Rule 8 — Ballet Teacher Preference

When multiple ballet slots are available, prioritise in this order:

| Priority | Slot | Why |
|----------|------|-----|
| Best | Sun 16:00 ADC | Primary ballet slot — **no pianist** (but best teacher) |
| Strong | HJS (Tue/Thu/Fri 09:30) | Floating — confirm teacher weekly |
| Strong | Thu 17:30 / Fri 17:30 ADC | Same teacher as Sun, but weaker accompanist; reliable backup |
| Backup | Mon 19:00 ADC | Backup when Sun is blocked — **has pianist**; rarely used |
| Rarely | Wed 17:30 ADC | Love the class; rarely make it due to office hours |
| Rarely | Sat 13:00 / 16:00 ADC | Usually at partner's; exceptions only |
| Rarely | Sun 10:45 / 11:30 | Same as above |

**Rule:** Always protect Sun 16:00 ADC as the primary slot. If Sun is blocked, Mon 19:00 ADC is the backup (note: Mon has pianist, Sun does not). If blocked further, move to Thu 17:30 ADC or Fri HJS (if teacher confirmed).

| Reformer Pilates | 2026-05-28 | 2026-06-25 | ● Done Sun Jun 21 — new deadline Jul 19 |

**How to apply:**
1. At each weekly planning session, check this table before building the week.
2. If a class deadline falls within the next 7 days → it must be slotted that week (unless overridden with a reason).
3. After a class is done, update the "Last Done" date and recalculate the deadline (+28 days).
4. Override is allowed: if you explicitly say "skip Mysore another week" that's fine — just note it. The rule flags; you decide.

- Iyengar: Mon 20:15 (default anchor) or Wed 18:30 (on non-choreo weeks)
