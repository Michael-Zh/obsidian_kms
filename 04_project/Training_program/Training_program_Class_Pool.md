# Class Pool — Danseur Noble Training Program
_Last updated: 2026-05-22_

## How to Use This File

This is the reference pool for scheduling and rescheduling. Use it when:
- Building the baseline weekly schedule
- Rescheduling around life events (dinners, shows, travel)
- Flagging intensity conflicts before confirming a week

**Intensity tags:**
- 🔴 High — significant cardiovascular and/or muscular demand; triggers energy crash risk if stacked
- 🟡 Moderate — active but manageable; can follow a High day
- 🟢 Light — recovery-oriented; can always be done

**Priority tiers:**
- **Primary** — default choice; design the week around these
- **Backup** — use when primary conflicts with life; not a default
- **Floating** — HJS classes: available Tue/Thu/Fri but teacher-dependent; confirm week by week. **Professional studio** — all classes at professional level; primary value is technical development and community connection
- **Rarely** — don't include in regular scheduling; flag only when specifically motivated
