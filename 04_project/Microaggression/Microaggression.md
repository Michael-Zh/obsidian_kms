# Microaggression — Response Database

## Goal

Build a personal reference for identifying and responding to anti-Asian microaggression and discrimination. Starts as private learning and pattern-collection. Can evolve into something others can learn from or contribute to once it has enough structure and substance.

## Scope

- **Primary**: Personal learning — understand patterns, build response toolkit, process experiences
- **Future**: Shareable/contributable once the structure is solid enough to be useful to others

## Coverage Areas

- **Recognition** — how to identify microaggression vs. accidental ignorance vs. systemic patterns
- **Response tactics** — field-tested approaches by context (street, workplace, social, online)
