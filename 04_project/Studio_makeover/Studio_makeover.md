# Home Improvement Project: Renovation Roadmap

**Goal:** Create a "Modern Art Museum Cafe / Neo-Industrial" aesthetic that supports a busy lifestyle, sports, and hosting in 33sqm.

**Style Principles:**

- **Curation:** "If it's not beautiful or functional, it is evicted." (Museum Rule).
    
- **The Vibe:** "MOMA meets Third Wave Coffee." Clean lines, white space, black steel accents, warm wood/plants.
    
- **Strategy:** Prioritize layout and flow first (Phase 1), then storage systems (Phase 2), then aesthetics (Phase 3).
    

## 🗓 Current Focus: The "Sanitation Runway"

- **Status:** Transitioning from Phase 1 (Layout) to Phase 2 (Systems).
    
- **Immediate Goal:** Execute Phase 1.5 (below) to maximize ROI on incoming professional cleaner.

## ♾️ Post-Clean Ongoing Maintenance & Dust Control

_Reinstate these habits once the baseline is restored to protect the clean space._

- **The Studio Dust Defense (Option C):**
    
    - _Action:_ Purchase and run a HEPA Air Purifier (e.g., IKEA STARKVIND, Xiaomi).
        
    - _Why:_ Studio apartments suffer from high textile shedding (bed/clothes in the same room). Purifiers catch the fibers _before_ they land on the HAY shelves.
        
- **The Sales Pipeline (Clothes & Items)**
    
    - _Action:_ Sort items immediately.
        
    - **The "€5 Rule":** If estimated selling price is **< €5** -> DONATE immediately. Do not list.
        
    - _Rule:_ If > €5, list on Vinted/Marktplaats. If not sold in 2 weeks -> Donate.
        
- **Daily Ritual:**
    
    - _Action:_ Clear the "Hot Surfaces" (Chair, TV Console, Kitchen Counter) before bed.
