# Cleaner Cleaning List

---

## 一、日常清洁（每次必做）

每次 cleaner 来都做的固定内容，预计 **约 60–75 分钟**。

### 洗手间

1. 地板：吸尘 + 拖干净
2. 所有表面（surfaces）：全部擦一遍
3. 坐厕：用清洁剂清洁
4. 镜子：擦干净
5. 柜子表面：擦拭

### 厨房

1. 所有柜子表面：擦拭
2. 灶台 + 台面（counter）：擦拭
3. 水池 + 水龙头：清洁
4. 抽油烟机表面：擦拭
5. 开放式置物架（杯子和调料）：擦拭
6. 外部电器（空气炸锅、咖啡机、高压锅等）：擦拭表面

### 全屋

1. 所有地板：先吸尘，再拖地
2. 地毯 + 沙发：用特殊吸头吸尘
3. shelf（架子）：掸掉灰尘（不用布擦）
4. 桌面：擦拭
5. 窗台台面：擦拭

---

## 二、深度清洁（逐步 catch up + 定期轮换）

以下项目是陈年积攒的内容，以及需要定期深度清洁的部分。建议每次日常清洁时选 **1–2 项** 加做，直到全部 catch up 后转为每季度一次。

### 洗手间

- 墙面 + 瓷砖水渍清洁
- 通风管道灰尘清理

### 厨房

- 柜子抽屉内部（尤其调料架）清洁
- Oven 清洁
- 冰箱清洁（可能需要自己 schedule）

### 全屋

- 高处表面清洁（电视顶上等）
- 展示书架其他隔层擦拭（等整理完东西后）
- 植物置物架周围区域（至少吸尘）
- 吊灯表面擦拭

## 四、荷兰语版 — Schoonmaakschema voor de Schoonmaker

### Wekelijkse Schoonmaak (elke keer verplicht)

**Ongeveer 60–75 minuten**

#### Badkamer
1. Vloer: stofzuigen + dweilen
2. Alle oppervlakken: afnemen
3. Toilet: reinigen met schoonmaakmiddel
4. Spiegel: schoonmaken
5. Kastdeuren / kastoppervlak: afnemen

#### Keuken
1. Alle kastdeuren / kastoppervlakken: afnemen
2. Fornuis + aanrecht: afnemen
3. Spoelbak + kraan: schoonmaken
4. Afzuigkap (buitenkant): afnemen
5. Open schap (kopjes & kruiden): afnemen
6. Apparaten (airfryer, koffiezetapparaat, snelkookpan): buitenkant afnemen

#### Hele Woning
1. Alle vloeren: eerst stofzuigen, dan dweilen
2. Tapijt + bank: stofzuigen met speciaal mondstuk
3. Planken / schappen: afstoffen (niet met een doek)
4. Bureau: afnemen
5. Vensterbank: afnemen

---

### Dieptereiniging (extra, elke sessie 1–2 items)

- **Badkamer:** muren + tegels (kalkvlekken), ventilatiekanaal stof
- **Keuken:** lades (vooral kruidenlade), oven, koelkast (zelf inplannen)
- **Hele woning:** hoge oppervlakken (bovenkant tv), boekenkast andere vakken, plantenplank omgeving, hanglamp

---

### Schema per Week

#### Week 1 — Dagelijkse Schoonmaak + Badkamer Dieptereiniging

| Onderdeel     | Wat te doen                                 |
| ------------- | ------------------------------------------- |
| **Verplicht** | Badkamer + Keuken + Hele woning             |
| **Extra**     | Badkamermuren + tegels kalkvlekken reinigen |
|               | Ventilatiekanaal stof verwijderen           |
| Totaal        | ~1,5 uur                                    |

#### Week 3 — Dagelijkse Schoonmaak + Keuken Dieptereiniging

| Onderdeel | Wat te doen |
|-----------|-------------|
| **Verplicht** | Badkamer + Keuken + Hele woning |
| **Extra** | Lades schoonmaken (vooral kruidenlade) |
| | Oven schoonmaken |
| Totaal | ~1,5 uur |

#### Week 5 — Dagelijkse Schoonmaak + Hoge Oppervlakken

| Onderdeel | Wat te doen |
|-----------|-------------|
| **Verplicht** | Badkamer + Keuken + Hele woning |
| **Extra** | Hoge oppervlakken (bovenkant tv, etc.) |
| | Hanglamp afnemen |
| Totaal | ~1,5 uur |

#### Week 7 — Dagelijkse Schoonmaak + Boekenkast & Planten

| Onderdeel | Wat te doen |
|-----------|-------------|
| **Verplicht** | Badkamer + Keuken + Hele woning |
| **Extra** | Boekenkast andere vakken afnemen |
| | Plantenplank omgeving schoonmaken (afnemen of stofzuigen) |
| Totaal | ~1,5 uur |

#### Daarna (wanneer alles is bijgewerkt)

- **Reguliere schoonmaak:** elke keer ~1 uur
- **Dieptereiniging:** 1 keer per kwartaal
- **Koelkast:** zelf inplannen wanneer nodig
